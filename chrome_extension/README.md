# 英语学习助手 Chrome 插件

## 安装步骤

### 1. 修复插件配置
已修复 `manifest.json` 文件，移除了对不存在文件的引用，确保插件能够正常安装。

### 2. 安装插件到 Chrome 浏览器

1. 打开 Chrome 浏览器
2. 进入扩展程序页面：`chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」按钮
5. 选择 `chrome_extension` 文件夹
6. 插件应该会成功加载，显示在扩展程序列表中

### 3. 启动本地服务器

在使用插件之前，需要启动本地 API 服务器：

1. 打开命令行终端
2. 进入项目目录：`cd d:\code\Englishlearning`
3. 激活虚拟环境：`venv\Scripts\activate`
4. 安装依赖：`pip install -r requirements.txt`
5. 启动服务器：`python api_server.py`
6. 服务器应该在 `http://localhost:5000` 运行

## 使用方法

### 1. 基本功能
- **查询单词**：在视频中选中单词，点击弹出菜单或按 `Ctrl+Q`
- **添加联想记忆**：选中单词后按 `Ctrl+M`，填写联想记忆内容
- **添加到生词本**：通过弹出菜单添加单词到生词本

### 2. 支持的网站
- YouTube
- Bilibili
- Netflix
- TED

### 3. 服务器状态
- 插件会自动检查本地服务器连接状态
- 如果服务器未启动，插件会进入本地模式，部分功能受限

## 常见问题

### Q: 插件安装失败
A: 确保已按照上述步骤修复了 `manifest.json` 文件，并且所有引用的文件都存在。

### Q: 服务器连接失败
A: 确保本地服务器已启动，并且在 `http://localhost:5000` 运行。

### Q: 部分功能不可用
A: 检查服务器是否正常运行，以及网络连接是否稳定。

## 功能特点

- 🎯 快速查询单词，支持多种在线词典
- 💡 自动生成联想记忆，提高记忆效率
- ✏️ 支持添加自定义联想记忆
- 📊 学习统计和复习提醒
- 🎥 视频网站集成，支持字幕单词提取

## 技术说明

- **插件类型**：Chrome 扩展（Manifest V3）
- **核心功能**：单词查询、联想记忆、学习记录
- **API 服务**：本地 Flask 服务器
- **数据存储**：SQLite 数据库

---

**提示**：为了获得最佳体验，建议在使用插件时保持本地服务器运行。
